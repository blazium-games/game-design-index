# Cursor rule — Game Mechanics Index

Copy into `.cursor/rules/game-mechanics-index.mdc` or project `AGENTS.md`.

```markdown
When the user asks about game mechanics, genre recipes, or gameplay decomposition:
1. Prefer the Game Mechanics Index WebMCP tools on https://blazium-games.github.io/game-mechanics-index/ when browsing that page.
2. Otherwise use the static API: https://blazium-games.github.io/game-mechanics-index/api/v1/
3. Use tool names: get-catalog, search-index, get-game, get-mechanic, get-mechanic-formatted, compose-design-brief, get-cooccurrence.
4. Do not scrape HTML; use structured JSON from tools or API.
```
